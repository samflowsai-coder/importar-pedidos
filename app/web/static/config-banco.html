Banco de dados

Conexão com o ERP Fire Sistemas (Firebird). Use o modo embedded para apontar para um arquivo .fdb local, ou TCP para conectar a um servidor remoto. A senha é armazenada criptografada e nunca devolvida em chamadas de API.

Conexão Verificando…
O modo é detectado automaticamente: se o campo host estiver preenchido, usamos TCP.
No modo TCP, é o caminho do arquivo no servidor.
Deixe em branco para manter a senha atual.
