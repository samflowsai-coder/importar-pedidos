Seção 1

Pré-requisitos

Antes de instalar, o computador precisa ter o Python instalado. É um programa gratuito, feito uma única vez.

1

Abrir o site do Python

Acesse python.org/downloads no navegador e clique no botão amarelo de download (versão 3.11 ou superior).

2

Executar o instalador

Abra o arquivo baixado. Importante: na primeira tela do instalador, marque a opção Add Python to PATH antes de clicar em "Install Now".

3

Confirmar a instalação

Abra o Prompt de Comando (tecla Windows → digite cmd → Enter) e execute python --version. Se aparecer um número de versão, está pronto.

⚠️ Se já tiver Python instalado: pule esta seção. O sistema funciona com qualquer versão 3.9 ou superior.

Seção 2

Instalação do sistema

Com o Python instalado, a instalação do Importar Pedidos é feita em um único clique.

1

Copiar a pasta do sistema

Copie a pasta importar pedidos para um local fixo no computador — por exemplo, C:\SamFlowsAI\importar pedidos. Não coloque na Área de Trabalho.

2

Executar o instalador

Dentro da pasta, dê duplo-clique em instalar.bat. Uma janela preta vai abrir e mostrar o progresso. Aguarde até aparecer a mensagem "Instalação concluída com sucesso".

3

Criar atalho na Área de Trabalho

Clique com o botão direito em iniciar.bat → Enviar para → Área de Trabalho (criar atalho). É por este atalho que o operador abrirá o sistema todo dia.

✅ O instalar.bat precisa ser executado apenas uma vez. A partir daí, o operador usa somente o iniciar.bat.

Seção 3

Configuração inicial

3.1 — Chave de API

O sistema utiliza inteligência artificial como recurso auxiliar. Para isso, é necessário configurar uma chave de acesso no arquivo .env.

Abra a pasta do sistema, clique com o botão direito no arquivo .env → Abrir com → Bloco de Notas.

# Substitua pelo valor fornecido pelo suporte
OPENROUTER_API_KEY=sua-chave-aqui
⚠️ Não compartilhe este arquivo nem envie a chave por e-mail ou WhatsApp. Em caso de dúvida, entre em contato com o suporte.

3.2 — Pastas de trabalho

Após abrir o sistema pela primeira vez, clique no ícone de configurações (engrenagem, canto superior direito) e defina:

Campo O que configurar Exemplo
Pasta de entrada Onde os PDFs e planilhas dos fornecedores serão colocados C:\Pedidos\Entrada
Pasta de saída Onde os arquivos prontos para o ERP serão salvos C:\Pedidos\ERP
💡 Crie as pastas antes de configurar. A pasta de entrada pode ser a mesma onde hoje os pedidos são salvos ao serem recebidos por e-mail.

3.3 — Conexão com a internet

O sistema funciona sem internet para os fornecedores já cadastrados (Riachuelo, Centauro, Beira Rio, Kolosh, Sam's Club, etc.). A internet é necessária apenas para processar formatos novos ou desconhecidos, que são enviados para análise automática.

Seção 4

Uso diário

O fluxo de trabalho é simples e acontece inteiramente no navegador.

1

Iniciar o sistema

Dê duplo-clique no atalho Importar Pedidos da Área de Trabalho. O navegador abre automaticamente com o sistema. A janela preta que aparece não deve ser fechada.

2

Colocar os pedidos na pasta de entrada

Salve os arquivos PDF ou XLS recebidos dos fornecedores na pasta de entrada configurada. Podem ser vários arquivos de uma vez.

3

Selecionar e importar

Na aba Pendentes, clique em Atualizar para ver os arquivos. Marque os que deseja processar e clique em Importar. O sistema gera os arquivos XLS para o ERP automaticamente.

4

Baixar e importar no ERP

Após o processamento, os links de download aparecem na tela. Clique para baixar cada arquivo XLS e importe normalmente no ERP. Os pedidos originais são movidos automaticamente para a pasta Pedidos importados.

5

Encerrar o sistema

Feche a janela preta (ou pressione Ctrl+C dentro dela) quando terminar o uso do dia.

Aba Importados

A aba Importados mostra o histórico de todos os pedidos processados — com data, número do pedido, cliente e links para baixar novamente os arquivos gerados.

Caso precise reprocessar um pedido (por exemplo, se o arquivo do ERP for perdido), clique em Reimportar na entrada correspondente.

Seção 5

Dúvidas frequentes

O navegador não abriu automaticamente

Abra o navegador manualmente e acesse http://localhost:3636.

Apareceu "Formato não reconhecido"

O arquivo é de um fornecedor ainda não cadastrado no sistema. Envie o arquivo para o suporte para que seja adicionado. O sistema já cobre: Riachuelo, Centauro, Beira Rio, Kolosh / Dakota, Sam's Club, Kallan, Magic Feet, Authentic Feet e NBA.

O sistema ficou lento ou travou

Feche a janela preta e abra o sistema novamente pelo atalho da Área de Trabalho.

Preciso processar um pedido de novo

Acesse a aba Importados, encontre o pedido e clique em Reimportar.

Esqueci qual pasta está configurada

Clique na engrenagem (canto superior direito) para ver e alterar as pastas configuradas.

Atualizações do sistema

Quando houver atualização, o suporte vai enviar uma nova versão da pasta. Substitua a pasta antiga pela nova e execute instalar.bat novamente. As configurações e histórico de importações são preservados.
